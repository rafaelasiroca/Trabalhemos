    🎨 Sistema de Design - Trabalhemos
Neste projeto, utilizamos um framework UI simples com pequenas personalizações para refletir uma identidade acessível, amigável e prática.

    1. Base da Estrutura

Framework escolhido: Bootstrap
Motivação: Permite desenvolvimento rápido, é fácil de usar e já possui componentes prontos ideais para um site simples de divulgação de serviços.

    2. Paleta de Cores (Personalização)

As cores foram escolhidas para transmitir confiança, simplicidade e proximidade entre trabalhadores e clientes:

Cor Primária (Destaque): #2E7D32 (Verde escuro)
Uso: Botões principais, links e ações importantes. Representa trabalho, crescimento e confiança.
Cor Secundária (Apoio): #1565C0 (Azul médio)
Uso: Destaques secundários e elementos interativos. Transmite segurança.
Cor de Fundo: #F5F5F5 (Cinza claro)
Uso: Fundo geral das páginas para manter leveza visual.
Cor de Texto: #212121 (Cinza escuro)
Uso: Textos principais para boa legibilidade.
Cor de Sucesso: #4CAF50 (Verde claro)
Uso: Mensagens positivas (ex: “Perfil atualizado com sucesso”).

    3. Tipografia

Utilizada via Google Fonts para manter um visual moderno e acessível:

Títulos (H1 a H6): Poppins, sans-serif (Peso: 600)
Textos e Formulários: Open Sans, sans-serif (Peso: 400)

    4. Diretrizes de Uso de Componentes

Regras simples para manter consistência visual no sistema:

Botões (.btn / .btn-primary):
Ações principais (ex: cadastrar, salvar perfil) usam cor primária (verde).
Ações secundárias usam contorno ou cor neutra.
Cartões (.card):
Usados para exibir perfis de trabalhadores.
Devem conter nome, descrição curta e botão “Ver mais”.
Formulários (.form-control):
Campos simples e objetivos.
Apenas os dados essenciais devem ser obrigatórios (Nome, CPF/CNPJ).
Listas de Perfis:
Devem ser organizadas em formato de cards ou lista simples.
Priorizar leitura rápida e escaneável.

    5. Princípios de Design
Simplicidade: Interface direta, sem excesso de informação.
Acessibilidade: Fácil de usar para qualquer tipo de usuário.
Clareza: Informações objetivas sobre os serviços oferecidos.
Rapidez: O usuário deve conseguir se cadastrar e divulgar seu trabalho rapidamente.
