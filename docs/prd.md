📄 Documento de Requisitos do Produto (PRD) - Trabalhemos

1. Visão Geral e Objetivo
   Trabalhemos é um site web que proporciona a divulgação de serviços autonomos, como diarista,manicure e pedreiro.
   O diferencial do Trabalhemos está na forma simples, acessível e direta de conectar pessoas a profissionais autônomos da sua região.
   A plataforma foi pensada para valorizar trabalhadores como diaristas, manicures, pedreiros e muitos outros, oferecendo visibilidade real sem burocracia e sem a necessidade de conhecimentos técnicos.

2. Atores do Sistema:
   TRABALHADOR: usuário que realiza seu cadastro na plataforma, tornando seu perfil público com informações essenciais como nome,
    telefone e descrição dos serviços oferecidos, facilitando o contato direto com possíveis clientes.

3. Histórias de Usuário e Escopo:
   👤 Épico 1: Cadastro e Perfil do Trabalhador
US01 - Cadastro de Trabalhador:
Como um Trabalhador, quero me cadastrar na plataforma informando meus dados (Nome, CPF/CNPJ), para criar meu perfil público.
   Critérios de Aceitação:
Todos os campos obrigatórios devem ser preenchidos (Nome, CPF ou CNPJ).
O CPF/CNPJ deve ser válido.
O sistema deve impedir cadastros duplicados.
US02 - Completar Perfil:
   Como um Trabalhador, quero adicionar uma descrição sobre meu trabalho e, opcionalmente, anexar um currículo, para apresentar melhor meus serviços.
Critérios de Aceitação:
A descrição deve permitir texto livre.
O currículo deve ser opcional.
O sistema deve aceitar upload de arquivo (ex: PDF).

   🧰 Épico 2: Divulgação de Serviços
US03 - Publicar Perfil Profissional:
Como um Trabalhador, quero que meu perfil fique visível na plataforma, para que clientes possam conhecer meus serviços.
Critérios de Aceitação:
   O perfil deve exibir nome, descrição e serviços.
   Apenas perfis cadastrados podem ser exibidos.
US04 - Editar Informações do Perfil:
   Como um Trabalhador, quero editar meus dados e descrição sempre que necessário, para manter minhas informações atualizadas.
Critérios de Aceitação:
O usuário deve conseguir atualizar suas informações.
As alterações devem ser refletidas imediatamente no perfil público.

   🔎 Épico 3: Busca e Visualização de Profissionais
US05 - Visualizar Lista de Trabalhadores:
Como um Visitante, quero visualizar uma lista de profissionais cadastrados, para encontrar alguém que atenda minha necessidade.
Critérios de Aceitação:
   A lista deve mostrar nome e breve descrição.
   A lista deve carregar corretamente mesmo com vários usuários cadastrados.
US06 - Ver Detalhes do Perfil:
Como um Visitante, quero acessar o perfil completo de um trabalhador, para entender melhor seus serviços.
Critérios de Aceitação:
Deve exibir nome, descrição e currículo (se disponível).
A navegação deve ser simples e intuitiva.

   📞 Épico 4: Contato com o Trabalhador
US07 - Visualizar Informações de Contato:
Como um Visitante, quero ver as informações de contato do trabalhador, para poder entrar em contato diretamente.
Critérios de Aceitação:
O perfil deve exibir telefone ou outra forma de contato.
As informações devem ser visíveis apenas no perfil do trabalhador.
